Final Year Project - Rendering Interactive Visualization in Real-time based on data derived from a Cloud Datacentre

For my Final Year Project I am creating a piece of exploratory software that generates bespoke interactive visualizations that displays information in real-time. The software aims to be able to process data is derived from a distributed system. This makes the data difficult to process due to the volume of the data, the velocity in which the data is sent, the heterogeneous in nature of the data and acceptable latency in relation to rendering the visualization to the user. Although I have regular meetings with my supervisor, the work is largely independent therefore requires good time management and planning skills.

Utilised Technology: Python, Flask, REST

Skills Demonstrated: Project planning, teamwork, interface design, testing, software engineering.

Arduino Weather Station

The aim of this project to use sensors connected on the Arduino to predict the weather. A combination of temperature and humidity sensors pass the data through serial communcation where it is deserialized. The weather is then computed on a Raspberry Pi before being posted to Twitter.

Utilised Technology: C++, Python, Arduino, Raspberry Pi.

Skills Demonstrated: Project planning, electronics, invoking web services.

Delete Generated Youtube Tweets

When linking social media accounts to websites such as Youtube, these websites will often tweet on the users behalf. If the user would like to disable this functioanlity the process of deleting these tweets can be quite time consuming. This project aims to delete tweets generated tweets, with the possiblity of extending this functionality to other sites in the future. My main goal with project is to gain a better understanding of Twitter's RESTful API's for the process of data mining and content generation.

Utilised Technology: Python, Rest

Skills Demonstrated: Project planning, invoking web services.
