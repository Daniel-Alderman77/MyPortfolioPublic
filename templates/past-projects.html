Orienteer

Orienteer was created in a group project as part of my degree. It is designed to help the user navigate the great outdoors. The compass view shows the direction of magnetic north, with the latitude and the longitude. The map view has uses Apple's Maps API to drop a pin to display the user current location map with a pin. The weather view displays information about the weather, such as temperature, wind speed, wind direction and the time for sunrise and sunset. It gets queries data from openweathermap.org's REST API and utilising the user's coordinates to provide wather information for their location.

Utilised Technology: Swift, REST, Apple Maps.

Skills Demonstrated: Teamwork, project planning, interface design, pair programming, invoking web services.

My Portfolio

My Portfolio was built using the Python micro web framework Flask. My design was implement using a combination Bootstrap and CSS3. It has been deployed using Amazon Webservices' (AWS) Elastic Beanstalk to allow server instance to scale as demand increases and decreases.

Gig Finder

GigFinder takes the user's Last.fm username then queries the last band this user has listened to then displays the next musical concert that this band is playing. This webapp uses the RESTful APIs provided by Songkick and Last.fm to obtain this information. The webapp is written in Java and utilises JSP to provide a user interface for enter their details and display the results. This webapp is designed to be packaged as a WAR file to be run in Apache Tomcat.

Utilised Technology: Java, JSP, REST, Apache Tomcat.

Skills Demonstrated: Project planning, invoking web services.
